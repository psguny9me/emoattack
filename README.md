# 🎮 Emoji Defense (이모지 디펜스)

현대적인 다크 테마 디자인과 이모지를 활용한 타워 디펜스 게임입니다.

![Game Preview](https://raw.githubusercontent.com/psguny9me/emoattack/main/preview.png) *(Note: Replace with actual image URL if available)*

## 🚀 주요 기능

### 🎯 핵심 시스템
- **다양한 타워**: 궁수(🏹), 머신건(🔫), 폭탄(💣), 레이저(⚡)
- **전략적 공격 로직**: 직선 발사, 유도탄, 즉시 타격 등 타워별 특화 로직
- **타워 업그레이드 (Merge)**: 같은 레벨의 동일 타워를 합쳐 더 강력한 타워로 진화
- **반응형 디자인**: PC와 모바일 환경 모두 최적화된 레이아웃 지원
- **현대적 UI**: 글래스모피즘(Glassmorphism)과 매끄러운 그라데이션 적용

### 👾 적 캐릭터
- **이동속도 밸런싱**: 플레이어가 충분히 전략을 세울 수 있도록 조정된 적들의 속도
- **웨이브 시스템**: 총 10단계의 웨이브와 강력한 보스 드래곤(🐉) 등장

## 🕹 조작법

1. **시작**: 화면 하단의 `▶️ 시작` 버튼을 눌러 게임을 시작합니다.
2. **타워 선택**: 오른쪽(또는 하단) 패널에서 원하는 타워를 클릭합니다.
3. **타워 배치**: 게임 화면의 빈 공간(그리드 점이 있는 곳)을 클릭하여 배치합니다.
4. **타워 합치기**: 같은 레벨의 타워를 드래그하여 다른 타워 위에 놓으면 레벨업됩니다.
5. **일시정지**: `⏸️ 일시정지` 버튼으로 게임을 잠시 멈출 수 있습니다.

## 🛠 기술 스택
- **HTML5 Canvas**: 고성능 게임 렌더링
- **Vanilla JavaScript**: 게임 엔진 및 로직 구현
- **CSS3**: 애니메이션 및 반응형 레이아웃 스타일링

## 🌐 온라인 플레이 방법

1. GitHub Repository의 **Settings > Pages** 메뉴로 이동합니다.
2. **Build and deployment > Branch**에서 `main` 브랜치를 선택하고 `Save`를 누릅니다.
3. 잠시 후 상단에 표시되는 URL을 통해 온라인에서 즉시 플레이할 수 있습니다!
